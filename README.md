## Uuidgen

A useful command-line tool for developers.

### Installation

```bash
git clone https://github.com/simplestar-992/uuidgen.git
cd uuidgen
go build -o uuidgen .
```

### Usage

```bash
./uuidgen --help
```

## Quick Start

```bash
./uuidgen [command]
```
