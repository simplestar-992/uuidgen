# UUIDGEN

**Useful CLI tool for developers**

## Features

- 🚀 Fast and efficient
- 📦 Single binary, no dependencies  
- 🔧 Easy to use
- 🔒 Secure by design

## Installation

```bash
git clone https://github.com/simplestar-992/uuidgen.git
cd uuidgen
go build -o uuidgen .
```

## Usage

```bash
./uuidgen --help
```
