## ✨ Features

- 🚀 Blazing fast performance
- 💎 Clean and intuitive API
- 🔧 Zero configuration needed
- 🛡️ Secure by default
- 📦 Single binary, no dependencies

## 📥 Installation

```bash
git clone https://github.com/simplestar-992/uuidgen.git
cd uuidgen
go build -o uuidgen .
./uuidgen --help
```

## 🚀 Quick Start

```bash
./uuidgen -h
```

## 🧪 Examples

```bash
# Example 1
./uuidgen status

# Example 2  
./uuidgen --verbose

# Example 3
./uuidgen help
```
